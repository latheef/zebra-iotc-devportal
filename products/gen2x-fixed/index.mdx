# IOTC for Gen2x Fixed RFID Readers

The Gen2x IoT Connector for Zebra next-generation fixed RFID readers extends the standard fixed reader capabilities with Gen2x (EPC Gen2v2) features including enhanced security, cryptographic authentication, and extended tag memory operations.

## Protocols

| Protocol | Purpose |
|----------|---------|
| **REST API** | Reader configuration, status queries, firmware management, Gen2x feature configuration |
| **MQTT** | Real-time tag data streaming (including Gen2x metadata), system events |

## Gen2x Enhancements

- **Cryptographic Tag Authentication**: Verify tag identity using AES-128 challenge-response
- **Extended Memory**: Access larger user memory banks on Gen2x-compliant tags
- **Untraceable**: Command tags to hide portions of their EPC or TID
- **Enhanced Security**: Encrypted tag-reader communication
