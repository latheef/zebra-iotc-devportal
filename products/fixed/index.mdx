# IOTC for Fixed RFID Readers

The Zebra IoT Connector for fixed RFID readers provides both a **REST API** for configuration and management, and **MQTT** for real-time telemetry and event streaming.

## Protocols

| Protocol | Purpose |
|----------|---------|
| **REST API** | Reader configuration, status queries, firmware management |
| **MQTT** | Real-time tag data streaming, system events, asynchronous commands |
