# Fixed Reader MQTT API Reference

This page renders the interactive AsyncAPI specification for the Zebra fixed RFID reader IoT Connector MQTT interface.

:::info
The interactive AsyncAPI reference is generated from the specification file at `specs/fixed/asyncapi.yml`.
:::

## Overview

| Channel | Direction | Description |
|---------|-----------|-------------|
| `{tenantId}/{deviceSerial}/command` | Publish | Send commands (start/stop inventory, configure) |
| `{tenantId}/{deviceSerial}/response` | Subscribe | Command responses |
| `{tenantId}/{deviceSerial}/data/tagReads` | Subscribe | Real-time tag read telemetry |
| `{tenantId}/{deviceSerial}/event/{eventType}` | Subscribe | System events |

Refer to the AsyncAPI specification for the full message schemas.
