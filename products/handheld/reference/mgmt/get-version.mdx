# get_version

Retrieves the firmware version, hardware model, serial number, and detailed component versions from the device.

## Topic

| | Topic |
|---|---|
| **Request** | `{tenantId}/MGMT/clients/cmnd/{deviceSerial}` |
| **Response** | `{tenantId}/MGMT/clients/resp/{deviceSerial}` |

## Direction

Application → Reader → Application

## Request Payload

```json
{
  "command": "get_version",
  "requestId": "abcd123"
}
```

| Field | Type | Required | Description |
|---|---|---|---|
| `command` | string | Yes | Must be `"get_version"` |
| `requestId` | string | Yes | Unique identifier to correlate the response |

## Response Payload

```json
{
  "command": "get_version",
  "requestId": "abcd123",
  "apiVersion": "V1.1",
  "readerVersion": {
    "firmwareVersion": "SAAFKS00-006-R02",
    "model": "RFD40",
    "serialNumber": "23053520102096",
    "sku": "RFD4031-G10B700-US",
    "companyName": "Zebra Technologies",
    "manufacturerName": "Zebra Technologies",
    "detailedVersions": {
      "scannerFirmware": "PAAEOC20-003-R01",
      "radioFirmware": "2.0.42.0",
      "iotcVersion": "V1.1"
    }
  },
  "response": {
    "code": 0,
    "description": "Success"
  }
}
```

| Field | Type | Description |
|---|---|---|
| `command` | string | Echo of the command name |
| `requestId` | string | Echo of the request identifier |
| `apiVersion` | string | API version: `"V1.0"` or `"V1.1"` |
| `readerVersion` | object | Device identity and version details |
| `readerVersion.firmwareVersion` | string | Overall firmware version string |
| `readerVersion.model` | string | Hardware model: `"RFD40"` or `"RFD90"` |
| `readerVersion.serialNumber` | string | Device serial number |
| `readerVersion.sku` | string | Full SKU part number |
| `readerVersion.companyName` | string | Always `"Zebra Technologies"` |
| `readerVersion.manufacturerName` | string | Always `"Zebra Technologies"` |
| `readerVersion.detailedVersions` | object | Component-level version breakdown |
| `readerVersion.detailedVersions.scannerFirmware` | string | Barcode scanner firmware version |
| `readerVersion.detailedVersions.radioFirmware` | string | RFID radio firmware version |
| `readerVersion.detailedVersions.iotcVersion` | string | IoT Connector version: always `"V1.1"` |
| `response` | object | Standard response envelope |
| `response.code` | integer | `0` for success |
| `response.description` | string | Human-readable result description |

## Error Responses

| Code | Description |
|---|---|
| `0` | Success |
| `1` | Failure |

## Notes

- This is a read-only, stateless query with no side effects.
- Use `get_version` after a firmware update (`set_os` + `reboot`) to confirm the new version is active.
- `readerVersion.detailedVersions.iotcVersion` determines which MQTT API commands and features are available.
