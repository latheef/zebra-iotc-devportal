# Get Installed Certificate

List all TLS certificates currently installed on the reader. The response includes certificate metadata such as issuer, subject, validity period, serial number, and SHA-256 fingerprint for each certificate. This is useful for auditing the device's certificate store and verifying that the correct certificates are deployed.

## Endpoint Details

| Property | Value |
|----------|-------|
| **Command** | `get_installed_certificate` |
| **Interface** | MGMT |
| **Direction** | PUB (command) → SUB (response) |
| **Command Topic** | `{tenantId}/MGMT/clients/cmnd/{deviceSerial}` |
| **Response Topic** | `{tenantId}/MGMT/clients/resp/{deviceSerial}` |
| **Content Type** | `application/json` |

## Request Payload (PUB)

```json
{
  "command": "get_installed_certificate",
  "requestId": "req-getcert-001"
}
```

### Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `command` | string | Yes | Must be `"get_installed_certificate"`. |
| `requestId` | string | Yes | Unique identifier to correlate the response. |

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `command` | string | Echoes the command name `"get_installed_certificate"`. |
| `requestId` | string | Echoes the request identifier. |
| `responseCode` | integer | Status code indicating the result. |
| `message` | string | Human-readable status message. |
| `certificates` | array | List of installed certificate objects. |
| `certificates[].name` | string | User-defined label for the certificate. |
| `certificates[].type` | enum | Certificate purpose: `mqtt`, `wifi`, or `client`. |
| `certificates[].issuer` | string | Certificate issuer distinguished name (DN). |
| `certificates[].subject` | string | Certificate subject. |
| `certificates[].validFrom` | date-time | Certificate validity start date (ISO 8601). |
| `certificates[].validTo` | date-time | Certificate validity end date (ISO 8601). |
| `certificates[].serialNumber` | string | Certificate serial number in hexadecimal. |
| `certificates[].fingerprint` | string | SHA-256 fingerprint of the certificate. |

## Response Codes

| Code | Description |
|------|-------------|
| `0` | Success |
| `1` | General error |
| `3` | Command not supported |
