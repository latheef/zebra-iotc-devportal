# Get Config

Retrieve the complete device configuration from a Zebra handheld RFID reader. This is a superset command that returns all configuration domains in a single response; equivalent to combining the output of `get_version`, `get_status`, `get_current_region`, `get_wifi`, and `get_eth`.

## Endpoint Details

| Property | Value |
|----------|-------|
| **Command** | `get_config` |
| **Interface** | MGMT |
| **Direction** | PUB (command) → SUB (response) |
| **Command Topic** | `{tenantId}/MGMT/clients/cmnd/{deviceSerial}` |
| **Response Topic** | `{tenantId}/MGMT/clients/resp/{deviceSerial}` |
| **Content Type** | `application/json` |

## Request Payload (PUB)

```json
{
  "command": "get_config",
  "requestId": "req-001"
}
```

### Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `command` | string | Yes | Must be `"get_config"`. |
| `requestId` | string | Yes | Unique identifier to correlate the response with this request. |

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `command` | string | Echo of the command name. |
| `requestId` | string | Echo of the request identifier. |
| `responseCode` | integer | Result code indicating success or failure. |
| `deviceConfig` | object | Container for the complete device configuration. |
| `deviceConfig.deviceVersion` | object | Hardware and software version information. See [Get Version](get-version). |
| `deviceConfig.deviceStatus` | object | Current operational status. See [Get Status](get-status). |
| `deviceConfig.regionInfo` | object | Regulatory region configuration. See [Get Current Region](get-current-region). |
| `deviceConfig.ethernetConfig` | object | Ethernet adapter configuration. See [Get Ethernet](get-eth). |
| `deviceConfig.wifiConfig` | object | Wi-Fi profiles and connection state. See [Get Wi-Fi](get-wifi). |
| `deviceConfig.endpointConfig` | object | Cloud endpoint connection configuration. |
| `deviceConfig.certificates` | array | Installed certificate entries. |
| `deviceConfig.eventConfig` | object | Event reporting and filtering configuration. |

## Response Codes

| Code | Description |
|------|-------------|
| `0` | Success. |
| `1` | General failure. |
| `3` | Device busy or unavailable. |
