# Delete Wi-Fi Profile

Remove a stored Wi-Fi network profile from a Zebra handheld RFID reader by specifying its SSID. If the device is currently connected to the network being deleted, the connection is terminated before the profile is removed.

## Endpoint Details

| Property | Value |
|----------|-------|
| **Command** | `delete_wifi_profile` |
| **Interface** | MGMT |
| **Direction** | PUB (command) → SUB (response) |
| **Command Topic** | `{tenantId}/MGMT/clients/cmnd/{deviceSerial}` |
| **Response Topic** | `{tenantId}/MGMT/clients/resp/{deviceSerial}` |
| **Content Type** | `application/json` |

## Request Payload (PUB)

```json
{
  "command": "delete_wifi_profile",
  "requestId": "req-001",
  "wifiConfig": {
    "ssid": "Warehouse-5G"
  }
}
```

### Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `command` | string | Yes | Must be `"delete_wifi_profile"`. |
| `requestId` | string | Yes | Unique identifier to correlate the response with this request. |
| `wifiConfig` | object | Yes | Wi-Fi profile identification. |
| `wifiConfig.ssid` | string | Yes | SSID of the Wi-Fi profile to delete. |

## Response Payload (SUB)

```json
{
  "command": "delete_wifi_profile",
  "requestId": "req-001",
  "responseCode": 0
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `command` | string | Echo of the command name. |
| `requestId` | string | Echo of the request identifier. |
| `responseCode` | integer | Result code indicating success or failure. |

## Response Codes

| Code | Description |
|------|-------------|
| `0` | Success. |
| `1` | General failure. |
| `15` | SSID not found, no profile exists with the specified SSID. |
| `17` | SSID missing; the required `ssid` field was not provided. |

## Example

### Request

```json
{
  "command": "delete_wifi_profile",
  "requestId": "delwifi-20260414-001",
  "wifiConfig": {
    "ssid": "Old-Guest-Network"
  }
}
```

### Response: Success

```json
{
  "command": "delete_wifi_profile",
  "requestId": "delwifi-20260414-001",
  "responseCode": 0
}
```

### Response: SSID Not Found

```json
{
  "command": "delete_wifi_profile",
  "requestId": "delwifi-20260414-001",
  "responseCode": 15
}
```
