# Get Status

Retrieve the current operational status of a Zebra handheld RFID reader. The response includes power source, radio activity state, battery health, internal temperature, NTP synchronization details, and terminal connection information.

## Endpoint Details

| Property | Value |
|----------|-------|
| **Command** | `get_status` |
| **Interface** | MGMT |
| **Direction** | PUB (command) → SUB (response) |
| **Command Topic** | `{tenantId}/MGMT/clients/cmnd/{deviceSerial}` |
| **Response Topic** | `{tenantId}/MGMT/clients/resp/{deviceSerial}` |
| **Content Type** | `application/json` |

## Request Payload (PUB)

```json
{
  "command": "get_status",
  "requestId": "req-001"
}
```

### Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `command` | string | Yes | Must be `"get_status"`. |
| `requestId` | string | Yes | Unique identifier to correlate the response with this request. |

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `command` | string | Echo of the command name. |
| `requestId` | string | Echo of the request identifier. |
| `responseCode` | integer | Result code indicating success or failure. |
| `deviceStatus` | object | Container for all device status fields. |
| `deviceStatus.powerSource` | string | Current power source. Enum: `BATTERY`, `USB`, `CRADLE`. |
| `deviceStatus.radioActivity` | string | RFID radio state. Enum: `ACTIVE`, `INACTIVE`. |
| `deviceStatus.radioConnection` | string | Radio connection state. Enum: `CONNECTED`, `DISCONNECTED`. |
| `deviceStatus.temperature` | integer | Internal device temperature in degrees Celsius. |
| `deviceStatus.systemTime` | string | Current system time in ISO 8601 date-time format. |
| `deviceStatus.ntp` | object | NTP synchronization details. |
| `deviceStatus.ntp.server` | string | Configured NTP server hostname. |
| `deviceStatus.ntp.offset` | number | Clock offset from NTP server in seconds. |
| `deviceStatus.ntp.reach` | integer | NTP reachability register value. |
| `deviceStatus.terminalConnection` | object | Terminal (host device) connection details. |
| `deviceStatus.terminalConnection.status` | string | Terminal connection state. Enum: `CONNECTED`, `DISCONNECTED`. |
| `deviceStatus.terminalConnection.type` | string | Terminal connection interface. Enum: `BLUETOOTH`, `USB`, `CIO`, `NONE`. |
| `deviceStatus.batteryStatus` | object | Battery health and charge details. |
| `deviceStatus.batteryStatus.stateOfHealth` | string | Battery health state. Enum: `GOOD`, `DEGRADED`, `REPLACE`. |
| `deviceStatus.batteryStatus.chargePercentage` | integer | Battery charge level, 0–100. |
| `deviceStatus.batteryStatus.chargeStatus` | string | Charge state. `0` = not charging, `1` = charging, `2` = fully charged. |
| `deviceStatus.batteryStatus.voltage` | integer | Battery voltage in millivolts (mV). |
| `deviceStatus.batteryStatus.temperature` | integer | Battery temperature in degrees Celsius. |

## Response Codes

| Code | Description |
|------|-------------|
| `0` | Success. |
| `1` | General failure. |
| `3` | Device busy or unavailable. |
