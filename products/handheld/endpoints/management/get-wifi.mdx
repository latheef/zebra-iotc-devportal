# Get Wi-Fi

Retrieve all configured Wi-Fi network profiles from a Zebra handheld RFID reader. The response includes each profile's SSID, security type, EAP method (for Enterprise networks), connection state, signal strength (when connected), and hidden network flag.

## Endpoint Details

| Property | Value |
|----------|-------|
| **Command** | `get_wifi` |
| **Interface** | MGMT |
| **Direction** | PUB (command) â†’ SUB (response) |
| **Command Topic** | `{tenantId}/MGMT/clients/cmnd/{deviceSerial}` |
| **Response Topic** | `{tenantId}/MGMT/clients/resp/{deviceSerial}` |
| **Content Type** | `application/json` |

## Request Payload (PUB)

```json
{
  "command": "get_wifi",
  "requestId": "req-001"
}
```

### Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `command` | string | Yes | Must be `"get_wifi"`. |
| `requestId` | string | Yes | Unique identifier to correlate the response with this request. |

## Response Payload (SUB)

```json
{
  "command": "get_wifi",
  "requestId": "req-001",
  "responseCode": 0,
  "wifiConfig": {
    "profiles": [
      {
        "ssid": "Warehouse-5G",
        "securityType": "WPA3_PERSONAL",
        "connected": true,
        "signalStrength": -42,
        "hidden": false
      },
      {
        "ssid": "Corporate-Secure",
        "securityType": "WPA3_ENTERPRISE",
        "eapMethod": "PEAP",
        "connected": false,
        "hidden": false
      }
    ]
  }
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `command` | string | Echo of the command name. |
| `requestId` | string | Echo of the request identifier. |
| `responseCode` | integer | Result code indicating success or failure. |
| `wifiConfig` | object | Container for Wi-Fi configuration data. |
| `wifiConfig.profiles` | array | List of configured Wi-Fi profiles. |
| `wifiConfig.profiles[].ssid` | string | Network SSID. |
| `wifiConfig.profiles[].securityType` | string | Security protocol. Enum: `WPA2_PERSONAL`, `WPA3_PERSONAL`, `WPA2_ENTERPRISE`, `WPA3_ENTERPRISE`, `OPEN`. |
| `wifiConfig.profiles[].eapMethod` | string | EAP method, present only for Enterprise security types. Enum: `PEAP`, `TLS`, `TTLS`. |
| `wifiConfig.profiles[].connected` | boolean | Whether the device is currently connected to this network. |
| `wifiConfig.profiles[].signalStrength` | integer | Signal strength in dBm. Present only when `connected` is `true`. |
| `wifiConfig.profiles[].hidden` | boolean | Whether the network is a hidden (non-broadcasting) SSID. |

## Response Codes

| Code | Description |
|------|-------------|
| `0` | Success. |
| `1` | General failure. |
| `3` | Device busy or unavailable. |
