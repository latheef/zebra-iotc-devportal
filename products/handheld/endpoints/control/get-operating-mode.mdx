# Get Operating Mode

Retrieve the current RFID operating mode configuration from the reader. The response includes the full operating mode settings such as RF profile, transmit power, access operations, triggers, query/select parameters, and tag metadata reporting options. If the reader cannot retrieve the configuration, the `operatingMode` object is omitted and a non-zero response code is returned.

## Endpoint Details

| Property | Value |
|----------|-------|
| **Command** | `get_operating_mode` |
| **Interface** | CTRL |
| **Direction** | PUB (command) → SUB (response) |
| **Command Topic** | `{tenantId}/CTRL/clients/cmnd/{deviceSerial}` |
| **Response Topic** | `{tenantId}/CTRL/clients/resp/{deviceSerial}` |
| **Content Type** | `application/json` |

## Request Payload (PUB)

Send a simple command payload with no additional configuration fields.

```json
{
  "command": "get_operating_mode",
  "requestId": "unique-id"
}
```

### Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `command` | string | Yes | Must be `"get_operating_mode"`. |
| `requestId` | string | Yes | Unique identifier to correlate request and response. |

## Response Payload (SUB)

On success, the response includes the full `operatingMode` object reflecting the reader's current configuration. On error code `3`, the `operatingMode` field is omitted.

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `command` | string | Echo of the command name. |
| `requestId` | string | Echo of the request identifier. |
| `apiVersion` | string | API version (e.g., `"V1.1"`). |
| `response.code` | integer | Numeric response code. |
| `response.description` | string | Human-readable description of the result. |
| `operatingMode` | object | Current operating mode configuration. Omitted when `response.code` is `3`. See [Set Operating Mode](set-operating-mode) for the full structure. |

## Response Codes

| Code | Description |
|------|-------------|
| `0` | SUCCESS |
| `1` | FAILURE |
| `3` | Unable to retrieve operating mode configuration |
