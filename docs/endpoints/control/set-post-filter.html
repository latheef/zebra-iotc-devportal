Set Post Filter

Configure tag data post-filters on the reader. Post-filters are applied after tag reads and allow you to include or exclude tags based on EPC prefix, suffix, regular expression patterns, RSSI threshold, or explicit whitelist/blacklist of tag identifiers.

Endpoint Details​

PropertyValue
Commandset_post_filter
InterfaceCTRL
DirectionPUB (command) → SUB (response)
Command Topic{tenantId}/CTRL/clients/cmnd/{deviceSerial}
Response Topic{tenantId}/CTRL/clients/resp/{deviceSerial}
Content Typeapplication/json

Request Payload (PUB)​

Publish a JSON message with the post-filter configuration. The fields required depend on the filterType selected.

{
  "command": "set_post_filter",
  "requestId": "unique-id",
  "postFilter": {
    "filterType": "PREFIX",
    "action": "INCLUDE",
    "pattern": "E200",
    "enable": true
  }
}

Fields​

FieldTypeRequiredDescription
commandstringYesMust be "set_post_filter".
requestIdstringYesUnique identifier to correlate request and response.
postFilterobjectYesPost-filter configuration object.
postFilter.filterTypeenumYesType of filter. One of: PREFIX, SUFFIX, REGEX, RSSI, WHITELIST, BLACKLIST.
postFilter.actionenumYesFilter action. One of: INCLUDE, EXCLUDE.
postFilter.patternstringConditionalMatch pattern (max 64 characters). Required when filterType is PREFIX, SUFFIX, or REGEX.
postFilter.rssiThresholdintegerConditionalRSSI threshold in dBm (range: -100 to 0). Required when filterType is RSSI.
postFilter.tagListarray of stringsConditionalList of tag EPC identifiers. Required when filterType is WHITELIST or BLACKLIST.
postFilter.enablebooleanYesEnable or disable the post-filter.

Filter Type Usage​

filterTypeRequired FieldsDescription
PREFIXpatternMatch tags whose EPC starts with the given pattern.
SUFFIXpatternMatch tags whose EPC ends with the given pattern.
REGEXpatternMatch tags whose EPC matches the regular expression.
RSSIrssiThresholdFilter tags by minimum RSSI signal strength.
WHITELISTtagListOnly include/exclude tags in the provided list.
BLACKLISTtagListOnly include/exclude tags in the provided list.

Example​

Request: Prefix Filter​

{
  "command": "set_post_filter",
  "requestId": "req-005",
  "postFilter": {
    "filterType": "PREFIX",
    "action": "INCLUDE",
    "pattern": "E200",
    "enable": true
  }
}

Response: Success​

{
  "command": "set_post_filter",
  "requestId": "req-005",
  "apiVersion": "V1.1",
  "response": {
    "code": 0,
    "description": "SUCCESS"
  }
}

Request: RSSI Filter​

{
  "command": "set_post_filter",
  "requestId": "req-006",
  "postFilter": {
    "filterType": "RSSI",
    "action": "EXCLUDE",
    "rssiThreshold": -70,
    "enable": true
  }
}

Response: Success​

{
  "command": "set_post_filter",
  "requestId": "req-006",
  "apiVersion": "V1.1",
  "response": {
    "code": 0,
    "description": "SUCCESS"
  }
}

Request: Whitelist Filter​

{
  "command": "set_post_filter",

  "postFilter": {
    "filterType": "WHITELIST",
    "action": "INCLUDE",
    "tagList": [
      "E20068060000004014B4E81E",
      "E20068060000004014B4E82F",
      "E20068060000004014B4E830"
    ],
    "enable": true
  }
}

Response: Success​

{
  "command": "set_post_filter",
  "requestId": "req-007",
  "apiVersion": "V1.1",
  "response": {
    "code": 0,
    "description": "SUCCESS"
