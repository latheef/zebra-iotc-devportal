alerts

Threshold-based alert events published when device conditions cross configured thresholds. Covers battery, temperature, power source, network, and firmware alert conditions.

Topic​

Topic
Event{tenantId}/MGMT/clients/event/{deviceSerial}

Direction​

Reader → Application

Payload​

Battery Low Alert​

{
  "event": "alerts",
  "deviceSerial": "24042AR50033",
  "timestamp": "2026-05-14T14:22:05.000Z",
  "alertType": "BATTERY_LOW",
  "severity": "WARNING",
  "details": {
    "batteryLevel": 18,
    "threshold": 20,
    "powerSource": "BATTERY"
  }
}

Battery Critical Alert​

{
  "event": "alerts",
  "deviceSerial": "24042AR50033",
  "timestamp": "2026-05-14T15:10:30.000Z",
  "alertType": "BATTERY_CRITICAL",
  "severity": "CRITICAL",
  "details": {
    "batteryLevel": 4,
    "threshold": 5,
    "powerSource": "BATTERY"
  }
}

Power Source Change Alert​

{
  "event": "alerts",
  "deviceSerial": "24042AR50033",
  "timestamp": "2026-05-14T09:30:00.000Z",
  "alertType": "POWER_SOURCE_CHANGE",
  "severity": "INFO",
  "details": {
    "previousSource": "CRADLE",
    "currentSource": "BATTERY"
  }
}
FieldTypeDescription
eventstringAlways "alerts"
deviceSerialstringSerial number of the reporting device
timestampstringAlert timestamp in ISO 8601 format
alertTypestringType of alert (see table below)
severitystringINFO, WARNING, or CRITICAL
detailsobjectAlert-specific data (varies by alertType)

Alert Types​

Alert TypeSeverityTrigger Condition
BATTERY_LOWWARNINGBattery drops below batteryLow.thresholdPercent
BATTERY_CRITICALCRITICALBattery drops below batteryCritical.thresholdPercent
TEMPERATURE_HIGHWARNINGInternal temperature exceeds temperatureHigh.thresholdCelsius
NETWORK_DISCONNECTWARNINGActive network interface disconnects
POWER_SOURCE_CHANGEINFODevice switches between battery, cradle, or USB power

Details Fields by Alert Type​

Alert TypeFieldTypeDescription
BATTERY_LOWbatteryLevelintegerCurrent battery percentage
BATTERY_LOWthresholdintegerConfigured threshold
BATTERY_LOWpowerSourcestringCurrent power source
BATTERY_CRITICALbatteryLevelintegerCurrent battery percentage
BATTERY_CRITICALthresholdintegerConfigured threshold
BATTERY_CRITICALpowerSourcestringCurrent power source
TEMPERATURE_HIGHtemperaturenumberCurrent temperature in Celsius
TEMPERATURE_HIGHthresholdintegerConfigured threshold
NETWORK_DISCONNECTinterfacestringDisconnected interface (WIFI, ETHERNET)
NETWORK_DISCONNECTlastConnectedEssidstringSSID of the last connected Wi-Fi network
POWER_SOURCE_CHANGEpreviousSourcestringPrevious power source
POWER_SOURCE_CHANGEcurrentSourcestringNew power source

Notes​

  • Alerts are enabled and configured via config_events. Each alert type can be independently enabled or disabled.
  • Battery alerts are triggered only once when the threshold is crossed. They re-arm when the battery level rises above the threshold (hysteresis).
  • NETWORK_DISCONNECT alerts are published only if the device still has MQTT connectivity (e.g., switched from Wi-Fi to Ethernet, or reconnected before the alert was generated).
  • Alert thresholds persist across reboots.
