# Zebra IOTC Developer Portal

Welcome to the Zebra IoT Connector (IOTC) Developer Portal. This site provides comprehensive documentation for building applications that communicate with Zebra RFID readers over **REST APIs** and **MQTT**.

## Choose Your Product

| Product | Protocols | Description |
|---------|-----------|-------------|
| [**Handheld Readers**](/handheld) | MQTT | Zebra RFD40/RFD90 Series handheld RFID sleds |
| [**Fixed Readers**](/fixed) | REST + MQTT | Zebra fixed RFID readers with full REST and MQTT support |
| [**Gen2x Fixed Readers**](/gen2x-fixed) | REST + MQTT | Next-generation fixed readers with enhanced Gen2x capabilities |

## What You'll Find Here

- **Getting Started**: Prerequisites, authentication, and first steps
- **Conceptual Guides**: Architecture, protocols, and best practices
- **Interactive API References**: Explore REST endpoints with Scalar and MQTT topics with AsyncAPI
- **Troubleshooting**: Common issues and solutions
