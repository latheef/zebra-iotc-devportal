About Supported Hardware (RFD40 / RFD90 Series)

📘 EXPLANATION

Overview​

Zebra IoT Connector for Handheld RFID is supported on Zebra's current-generation UHF RFID reader sleds. These devices attach to or pair with Zebra Android mobile computers and provide enterprise-grade RFID read/write capabilities in a handheld form factor.

Supported Reader Models​

RFD40 Premium / Premium Plus​

The RFD40 Premium is Zebra's flagship handheld RFID reader sled, designed for demanding enterprise environments. The Premium Plus variant adds long-range reading capabilities.

  • RFID frequency: 865–928 MHz (UHF, region-configurable)
  • RFID standard: EPC Gen2 V2 (ISO 18000-63)
  • Max RF power: 30 dBm (1 W) conducted
  • Read range: Up to 6 m (Premium), up to 9 m (Premium Plus)
  • Connectivity: Bluetooth 5.2 (BLE + Classic)
  • Barcode scanning: Integrated SE55 1D/2D scan engine
  • Battery: 3,000 mAh rechargeable Li-ion
  • Trigger: Physical gun-style trigger + programmable side button
  • Form factor: Pistol-grip sled attachment

RFD40 Standard​

A cost-optimized variant for medium-range RFID applications.

  • RFID frequency: 865–928 MHz (UHF, region-configurable)
  • RFID standard: EPC Gen2 V2 (ISO 18000-63)
  • Max RF power: 27 dBm (500 mW) conducted
  • Read range: Up to 3 m
  • Connectivity: Bluetooth 5.2 (BLE + Classic)
  • Barcode scanning: Integrated 1D/2D scan engine
  • Battery: 3,000 mAh rechargeable Li-ion
  • Trigger: Physical trigger + programmable side button
  • Form factor: Pistol-grip sled attachment

RFD90​

The RFD90 is a next-generation ultra-rugged sled reader for high-volume inventory and asset tracking in harsh environments.

  • RFID frequency: 865–928 MHz (UHF, region-configurable)
  • RFID standard: EPC Gen2 V2 (ISO 18000-63)
  • Max RF power: 30 dBm (1 W) conducted
  • Read range: Up to 10 m
  • Connectivity: Bluetooth 5.3
  • Barcode scanning: Integrated SE58 scan engine
  • Battery: 3,600 mAh rechargeable Li-ion (hot-swappable)
  • Trigger: Ergonomic gun-style trigger + configurable side buttons
  • Form factor: Pistol-grip sled with enhanced ergonomics

Firmware Requirements​

IOTC functionality requires a minimum reader firmware version. Earlier firmware versions do not include the IOTC agent and cannot communicate via MQTT.

IOTC VersionMinimum FirmwareNotes
V1.03.10.27Initial IOTC release for RFD40 series
V1.13.12.10Adds RFD90 support, enhanced tag filtering, battery telemetry

To check the current firmware version, query the reader via the mgmt interface or use 123RFID Desktop.

Compatible Host Devices​

The reader sled pairs with Zebra Android mobile computers via Bluetooth. Supported host devices include:

  • TC21 / TC26: Value-tier touch computers
  • TC52 / TC57: Mid-tier enterprise touch computers
  • TC72 / TC77: Premium enterprise touch computers
  • EC50 / EC55: Enterprise computers
  • MC3300x: Rugged mobile computers

The host device must run the Zebra RFID Mobile App (or a custom application using the Zebra RFID SDK for Android) to establish the Bluetooth connection and proxy MQTT traffic to the broker over the host's Wi-Fi or cellular network connection.

Regional SKU Considerations​

Reader sleds are shipped with region-specific RF configurations. The configured frequency band affects available channels and maximum transmit power. Ensure the correct regional SKU is deployed:
