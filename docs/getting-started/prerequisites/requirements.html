Hardware & Software Requirements

📕 REFERENCE: Use this checklist to verify you have everything needed before beginning integration with the Zebra IoT Connector (IOTC) platform.

Hardware Requirements​

ComponentRequirementNotes
RFID Reader SledZebra RFD40 or RFD90UHF RFID handheld reader sled
Firmware Version≥ 3.10.27Required for IOTC MQTT connectivity
Host Mobile DeviceAndroid smartphone or tabletFor Bluetooth pairing with the reader sled
USB CableUSB-A to USB-C (or USB-C to USB-C)For initial reader bootstrap via 123RFID Desktop
Configuration PCWindows 10 or Windows 11Required for running 123RFID Desktop

Software Requirements​

SoftwarePurposeDownload
123RFID DesktopReader bootstrap and region configurationZebra Support Portal
MQTT ClientTesting and debugging MQTT communicationSee options below

Recommended MQTT Clients​

Choose one of the following MQTT clients for development and testing:

  • Eclipse Mosquitto CLI: Command-line tools (mosquitto_pub, mosquitto_sub) ideal for scripting and automation
  • MQTT Explorer: Desktop GUI client with topic visualization, excellent for initial exploration
  • MQTTX: Cross-platform GUI and CLI client with JSON formatting support

Network Requirements​

RequirementDetails
Wi-Fi Network2.4 GHz or 5 GHz Wi-Fi network accessible by the reader sled
Internet AccessOutbound MQTT connectivity to the IOTC broker
MQTT PortTCP port 8883 (MQTT over TLS) must be open for outbound connections
DNS ResolutionThe reader must be able to resolve the IOTC broker hostname

Note: The reader sled communicates with the IOTC platform exclusively over MQTT 3.1.1 with TLS encryption. Ensure your network firewall permits outbound traffic on port 8883.

IOTC Platform Credentials​

Before connecting a reader, you must have the following from the Zebra IoT Connector platform:

CredentialDescription
Tenant IDYour unique IOTC tenant identifier
MQTT Broker HostThe IOTC MQTT broker endpoint hostname
MQTT UsernameAuthentication username for broker connection
MQTT PasswordAuthentication password for broker connection
Device Serial NumberThe serial number printed on your reader sled (also available via 123RFID Desktop)

See How to Obtain IOTC Credentials & Tenant ID for instructions on acquiring these values.

Pre-Flight Checklist​

Use this checklist before starting the Quick Start Tutorial:

  • RFD40 or RFD90 reader sled in hand
  • Firmware version ≥ 3.10.27 confirmed
  • 123RFID Desktop installed on a Windows PC
  • USB cable available for initial bootstrap
  • Wi-Fi network name (SSID) and password known
  • IOTC tenant credentials obtained
  • MQTT client installed on your development machine
  • Host Android device available for Bluetooth pairing

Warning: If your reader firmware is below version 3.10.27, you must update it before proceeding. See How to Verify Reader Firmware Version for instructions.
