Zebra IoT Connector: Handheld RFID

Welcome to the official documentation for the Zebra IoT Connector MQTT API for RFD40 / RFD90 Series handheld RFID reader sleds.

This documentation covers everything from initial setup to production fleet operations, organized into seven parts following the Diátaxis framework.
